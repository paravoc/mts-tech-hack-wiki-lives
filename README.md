## ⚡ Быстрый старт (самое важное)

``` bash
docker compose up --build
```

После запуска:

-   🌐 Frontend: http://localhost:8501\
-   🔧 Backend (health check): http://localhost:3000/health

## 📌 О проекте

WikiLive --- платформа для совместной работы с документами в реальном
времени.

## 🧩 Основные возможности

-   📄 Wiki-редактор\
-   💬 Комментарии и обсуждения\
-   🧠 AI-подсказки\
-   🔄 Версионирование\
-   👥 Реалтайм коллаборация
